This is a modern web application built using a fast, reliable frontend stack. The project is set up for local development, easy maintenance, and smooth deployment.

Tech Stack

The project uses the following technologies:

Vite – Fast build tool and dev server

React – Component-based UI library

TypeScript – Type-safe JavaScript

Tailwind CSS – Utility-first styling

shadcn/ui – Prebuilt, accessible UI components

This setup keeps things simple, scalable, and future-proof.

Getting Started (Local Setup)

Make sure Node.js and npm are installed on your system. Use a stable Node version.

1. Clone the Repository
git clone <YOUR_GIT_URL>

2. Go to the Project Folder
cd <YOUR_PROJECT_NAME>

3. Install Dependencies
npm install

4. Start the Development Server
npm run dev


The app will start with hot reload enabled. Any code changes will reflect instantly in the browser.

Editing the Code

You can edit the project using any of the following methods:

Using Your Local IDE

Open the project in VS Code or any IDE of your choice. Make changes, commit, and push as usual.

Editing Directly on GitHub

Open the file you want to edit

Click the pencil (✏️) icon

Make your changes and commit them

Using GitHub Codespaces

Go to the repository main page

Click Code → Codespaces

Create a new Codespace

Edit, commit, and push directly from the browser

Deployment

The project can be deployed on any modern hosting platform that supports Vite/React builds.

General steps:

Build the project

npm run build


Deploy the generated dist folder to your hosting provider.
